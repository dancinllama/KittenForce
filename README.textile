Sample usage:

		@<c:puppy myHeight="200" myWidth="200" />@

		@<c:puppy myHeight="200" myWidth="200" darylShaber="true" />@

		@<c:puppy myHeight="400" myWidth="300" />@

		@<c:puppy myHeight="400" myWidth="300" applyToAll="true" />@

		@<c:puppy myHeight="400" myWidth="300" applyToAll="true" darylShaber="true" />@
